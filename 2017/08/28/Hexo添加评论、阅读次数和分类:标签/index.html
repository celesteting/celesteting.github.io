注册LeanCloud创建类并得到App ID以及App Key

在设置里面的应用Key中可以得到 App ID 以及 App Key 这两个参数

  1. 注册 LeanCloud
  2. 应用 -> 创建新应用 -> 创建
  3. 进入“应用”-> “存储”->“数据”->“创建Class”创建一个Counter的类
  4. 设置 -> 应用Key -> 复制 App ID 以及 App Key 在编辑 \themes\你的主题\ _config.yml 中相应位置输入即可。
